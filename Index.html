What components are part of the Von Neuman Architecture?

A central processing unit (CPU), memory, and input/output (I/O) interfaces.

What is user space?

User space is system memory allocated to running applications.

What is the shell?

In computing, a shell is a computer program which exposes an operating system's services to a human user or other program.

Where is the shell located?

The shell is located on the application layer. This is one of the seven layers of the OSI model

What is the main task of the shell?

The shell is responsible for the execution of all programs that you request from your terminal.
